benchmark-octane
================

Octane benchmark for Node.js

* This exports [Octane](http://code.google.com/p/octane-benchmark/) benchmark suites for [benchmark.js](http://benchmarkjs.com/).

How to use
----------

### GitHub

    % git clone https://github.com/dai-shi/benchmark-octane.git
    % cd benchmark-octane
